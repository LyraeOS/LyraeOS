# What is it?
LyraeOS is a 64bit operating system using the limine bootloader. It is designed to be unix-kinda-like os meaning that it will have somewhat similar features to other unix based operating systems

# Goals
## Current Goals
- Memory Allocator
- Scheduler
- Userspace

## Future Goals
- Gui
- ELF binaries
- Filesystem

## Current Progress
- Very basic shell
- Conways game of life
- Minesweeper
- Serpinksi triangles
